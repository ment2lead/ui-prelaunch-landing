# Ment to Lead — Prelaunch Landing

Static React landing page for Ment to Lead with English and Ukrainian locales.

## Stack

- React 19 + TypeScript
- Vite SSR prerender (static site generation)
- Tailwind CSS v4
- react-i18next

## Development

```bash
npm install
npm run dev
```

Visit `/en` or `/uk`.

## Build

```bash
npm run build
npm run preview
```

Static output is written to `dist/`.
