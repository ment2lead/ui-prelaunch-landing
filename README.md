# Ment2Lead — Prelaunch Landing

Static React landing page for Ment2Lead.

## Stack

- React 19 + TypeScript
- Vite SSR prerender (static site generation)
- Tailwind CSS v4
- react-i18next

## Development

```bash
npm install
npm run dev
```

Open the root URL (`/`).

## Build

```bash
npm run build
npm run preview
```

Static output is written to `dist/`.
